极其简单的使用方法

不懂编程也没关系，只要三步就能用上强大的工具。

01

找到你需要的工具

在下方的工具库中，搜索能解决你当下需求的小助手。比如："番茄钟"或"照片重命名"。

02

一键复制图纸

点击工具卡片，你会看到它的详细功能。直接点击大大的「一键复制图纸」按钮。

03

交给 AI 运行

把复制好的图纸，粘贴发给你的 AI 助手 (如 ChatGPT, Claude, 豆包等)，它就会马上为你运行这个工具！

发布你的 Vibe 工具

有了一个好点子？用 AI 写好了代码？只需三步就能发布到社区。

01

复制「AI 发布协议」

点击下方按钮，将发布协议复制到剪贴板。

02

粘贴给你的 AI 助手

打开 ChatGPT / Claude / 豆包，把协议粘贴进去，再告诉它你的工具信息。

03

AI 自动提交

AI 会自动生成文档、打包代码，并通过 GitHub API 帮你提交 PR。
