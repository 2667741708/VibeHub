# 番茄工作法锁屏助手

> 每工作25分钟，弹出全屏倒计时提醒你休息5分钟。

---

## 这个工具能帮你做什么？

你有没有发现，工作或学习时不知不觉就过了好几个小时，等回过神来已经腰酸背痛？这个番茄钟工具会在你每工作 25 分钟后，**强制弹出一个全屏红色大字倒计时**，提醒你必须停下来休息 5 分钟。

它不是那种你可以轻松关掉的小通知。它是一个「霸道总裁式」的提醒——全屏、置顶、无法忽视。休息时还会随机给你一句鼓励的话，比如「你已经很棒了，站起来走走吧」。

## 创作故事

### 灵感来源
我总是控制不住自己连续工作太久，各种番茄钟 App 的通知都被我秒关。我需要一个「暴力版」的提醒工具——直接把整个屏幕霸占掉，让我没法继续工作。

### AI 创作过程
- **使用的 AI 平台**：VibeHub 内置引擎
- **关键 Prompt**：
  > "我总是控制不住自己玩手机，每工作25分钟就弹出全屏提醒休息5分钟，屏幕上用巨大的红色字显示倒计时和一句鼓励的话"
- **迭代过程**：一轮 Prompt 即生成完整代码，后续微调了鼓励话术。

## 技术实现

### 文件说明
| 文件 | 作用 |
|------|------|
| `src/pomodoro.py` | 唯一的入口文件，包含计时器逻辑和全屏 UI |

### 核心逻辑
使用 Python 的 `tkinter` 库创建全屏窗口。主循环每 25 分钟触发一次 `show_rest_screen()`，弹出全屏置顶的深色背景 + 红色大字倒计时。

## 如何使用

### 环境要求
- Python 3.8+
- tkinter（大多数系统自带）

### 运行方式
```bash
python pomodoro.py
```
